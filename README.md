# 🏥 Health Insurance Cost Predictor (ML Web App)

An interactive Machine Learning web application that predicts health insurance costs based on user inputs such as age, income, BMI, smoking status, and medical history. Built using Streamlit and trained regression models.

## 📌 Project Overview

This project predicts health insurance charges using machine learning.  
Users enter personal, financial, and medical details, and the trained model returns an estimated insurance cost instantly.

---

## 🧠 Problem Statement

Insurance cost depends on multiple factors such as:

- Age  
- BMI  
- Smoking habits  
- Medical history  
- Income level  

The goal is to build a machine learning system that predicts insurance costs accurately based on these features.
